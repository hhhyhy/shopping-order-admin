# Shopping Order Admin

쇼핑몰 주문과 배송 상태를 관리하는 관리자용 웹 서비스입니다.

## 프로젝트 개요
쇼핑몰에서 발생하는 주문 정보를 관리하고 배송 상태를 변경할 수 있는 백엔드 중심 프로젝트입니다.
4학년 2학기 취업 준비생 포트폴리오 프로젝트를 가정하여 작성했습니다.

## 주요 기능
- 회원 정보 관리
- 상품 정보 관리
- 주문 내역 조회
- 주문 상태 변경
- 배송 상태 관리

## 사용 기술
- Java
- Spring Boot
- MySQL
- JPA
- Thymeleaf

## 학습 내용
- 회원, 상품, 주문 데이터 관계 설계
- 관리자 페이지 구성
- Spring MVC 구조 적용
- 데이터 조회 및 상태 변경 처리
